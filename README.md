# Credit Risk Prediction Dashboard

A Streamlit web application for exploring and predicting credit risk using machine learning. This project provides an interactive dashboard where users can view credit-related data, visualize patterns, and generate credit risk predictions through a simple browser-based interface.

Live App: https://crpdapps.streamlit.app

---

## Overview

The Credit Risk Prediction Dashboard is designed to help users better understand credit risk factors through data visualization and predictive modeling. The app is built with Python and Streamlit, making it easy to run locally or deploy online through Streamlit Cloud.

The dashboard uses common data science libraries such as pandas, scikit-learn, matplotlib, and Streamlit to process data, train or apply machine learning models, and display results in an interactive format.

---

## Features

- Interactive web-based dashboard using Streamlit
- Credit risk prediction using machine learning
- Data loading and preprocessing
- Visual analysis of credit-related variables
- User-friendly interface for exploring model outputs
- Browser-accessible hosted version through Streamlit Cloud
